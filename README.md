rust-ffmpeg
===========

FFmpeg bindings for rust

[![Build Status](https://travis-ci.org/mewlips/rust-ffmpeg.svg)](https://travis-ci.org/mewlips/rust-ffmpeg)

## How to build

```
$ git clone https://github.com/mewlips/rust-ffmpeg.git
$ cd rust-ffmpeg
$ cargo build
```
